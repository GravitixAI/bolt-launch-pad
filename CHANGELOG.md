# Changelog

All notable changes to Bolt Launch Pad will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-11-24

### Added
- Initial project setup based on electron-vite-react-boilerplate
- Electron 39.2.3 with Vite 7.2.4
- React 19.2.0 with TypeScript 5.9.3
- Tailwind CSS 4.1.17 with shadcn/ui
- SQLite database support with better-sqlite3
- Secure IPC communication
- Auto-updater support
- Dark/light theme support
- Production-ready configuration
